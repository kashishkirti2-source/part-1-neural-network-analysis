# Part 1: Neural Network Fundamentals and Training Behavior Analysis

## Objective
This project focuses on building and analyzing a feed-forward neural network model for customer churn prediction using Python and TensorFlow/Keras.

The assignment demonstrates:
- Dataset exploration
- Data preprocessing
- Neural network model building
- Model training and evaluation
- Hyperparameter experimentation
- Final reflection on neural network learning behavior

---

## Dataset
Dataset used:
- `customer_churn_nn.csv`

Target variable:
- `churn`

---

## Tasks Completed

### Task 1: Dataset Understanding
- Explored dataset structure
- Checked missing values
- Analyzed feature types
- Generated statistical summary
- Visualized target distribution

### Task 2: Data Preprocessing
- Encoded categorical features
- Scaled numerical data
- Split data into training and testing sets

### Task 3: Neural Network Model Building
- Built feed-forward neural network
- Added hidden layers and activation functions
- Compiled model using Adam optimizer

### Task 4: Training and Evaluation
- Trained neural network
- Evaluated using accuracy, loss, confusion matrix, and classification report
- Visualized training curves

### Task 5: Hyperparameter Experimentation
- Tested multiple configurations
- Compared model performance across experiments

### Task 6: Final Reflection
- Explained neural network concepts including weights, biases, activation functions, and learning rate behavior
